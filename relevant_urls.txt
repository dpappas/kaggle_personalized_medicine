If you have installed python

On linux like systems you only have to run the following commands:
pip install matplotlib
pip install tensorflow
pip install TFLearn
pip install keras

On Windows systems:

-Install Anaconda 64 bit, python 3.6 version from:
https://www.continuum.io/downloads

-Open Anaconda prompt
-conda create -n tensorPython3 python=3.5    (in 3.6 if 3.5 does not work)
-activate tensorPython3
-conda install numpy pandas jupyter notebook matplotlib
-conda install scipy h5py

pip install tensorflow
pip install TFLearn
pip install keras

-go to this link:
http://www.lfd.uci.edu/~gohlke/pythonlibs/#curses
Download curses-2.2-cp35-none-win_amd64.whl

-pip install  curses-2.2-cp35-none-win_amd64.whl
